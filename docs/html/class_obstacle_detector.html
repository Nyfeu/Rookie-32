Detailed Description

Gerencia a detecção de obstáculos usando um sensor ultrassônico HC-SR04.

Esta classe fornece métodos para medir distâncias e verificar a presença de obstáculos, disparando um callback opcional quando um obstáculo é detectado dentro de uma distância limite.
