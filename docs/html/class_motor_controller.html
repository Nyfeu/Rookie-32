Detailed Description

Gerencia o controle de dois motores DC conectados via uma ponte H.

Esta classe permite controlar a movimentação de um robô ou veículo com dois motores (esquerdo e direito) conectados a pinos GPIO específicos. Ela oferece métodos para mover para frente, para trás, virar e parar, além de uma função para lidar com comandos de joystick (x, y).

Lida com os comandos de movimentação baseados em valores X e Y (joystick-like).

Interpreta os valores X e Y para determinar a direção do movimento (frente, trás, esquerda, direita) e chama a função de movimentação correspondente. Possui uma zona morta (DEAD_ZONE) para evitar movimentos indesejados.
