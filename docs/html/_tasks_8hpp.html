Detailed Description

Inicialização e gerenciamento central das tasks do sistema.

Este arquivo é responsável por declarar os semáforos e iniciar todas as tasks do sistema FreeRTOS. Inclui os headers individuais de cada task e define a função createFreeRTOSTasks, que realiza a criação, sincronização e, se ativado, registro no watchdog timer.
