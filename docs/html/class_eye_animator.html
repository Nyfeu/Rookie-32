Detailed Description

Gerencia a animação e exibição de "olhos" em um display OLED.

Esta classe é responsável por inicializar o display OLED, definir e atualizar as emoções exibidas (que são representadas por animações de olhos), e desenhar os frames correspondentes na tela.
