Detailed Description

Define a estrutura de dados e os frames usados para animações de olhos e expressões faciais.

Este arquivo fornece uma estrutura para representar um quadro (frame) de uma animação facial, bem como conjuntos de frames pré-definidos para representar emoções específicas como felicidade, calma, tristeza e uma expressão "nerd".
